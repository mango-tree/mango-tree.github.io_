Django Tutorial 5 - Template

Apr 06, 2017 in Django

이전 글처럼 string으로만 요청을 처리할 수도 있지만, 모든 html 응답을 저런 식으로 처리하는 것은 매우 힘들기 때문에, 장고에서는 템플릿을 지원한다.
기본적으로 DjangoTemplates라는, 장고 전용 템플릿을 사용하며, 입맛에 따라 다른 템플릿으로도 변경 가능하다.

template 설정

템플릿 관련 설정은 settings.py에 있다.

BACKEND - 템플릿 엔진을 정할 수 있다. 장고에 내장되어 있는 템플릿은 django.template.backends.django.DjangoTemplates 와 django.template.backends.jinja2.Jinja2이다.
DIRS - 템플릿이 있는 디렉토리 위치를 정해준다.
APP_DIRS - 각 애플리케이션 안에 templates 디렉토리를 체크할지를 알려준다.
DIR 파라미터에 다음과 같은 값을 넣어주자

root 디렉토리에서 ‘templates’ 디렉토리를 나타낸다.
