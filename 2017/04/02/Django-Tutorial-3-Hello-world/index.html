Django Tutorial 3 - Hello, world!

Apr 02, 2017 in Django

이제 장고를 구동해보자. 장고에는 자체 미니 웹서버가 있어서 개발/디버깅 시에 쓰기 유용하다. 물론 개발용 웹서버이기 때문에, 실제 서비스 배포시에는 Nginx 등의 다른 웹 서버와 연결해서 쓰는 것이 좋다. 자세한 내용은 uWSGI 와 Python3을 이용하여 Nginx로 배포하기 참조.

Hello, world

장고를 실행해보기에 앞서, settings.py에서 ALLOWED_HOSTS를 지정해야 한다.

가상 환경으로 들어온 뒤에, django-admin으로 생성한 프로젝트 안으로 들어간 후 manage.py를 이용하여 켤 수 있다.

runserver를 실행시키면 다음과 같이 웹서버가 실행된다.

붉은 경고문은 현재 우리가 migration을 하지 않았기 때문에 뜨는 것이다. 후에 데이터베이스를 만질 일이 있을 때 해결할 것이다.
